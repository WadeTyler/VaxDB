
### Select Entries
entry select "key|all" "table_name"

### Remove a Model
remove model "model_name";

### Create a table
create table "table_name" -> "model_name" ;

### Add an Entry
entry add "table_name" -> "key" -> atb:"val", atb:"val";

### Remove an Entry
entry remove "table_name" -> "key";

### Reset Database
reset

Note: Spaces do not work inside quotes
